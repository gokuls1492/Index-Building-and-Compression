How to run:
-----------------------------------------------------------------------
1. Copy file - 'Indexing.py' into local directory
2. Type the following command:
python Indexing.py /people/cs/s/sanda/cs6322/Cranfield /people/cs/s/sanda/cs6322/resourcesIR/stopwords

Important: In case any import error occurs, please download nltk package using following command:
Type python in command prompt & then:
import nltk
nltk.download()
nltk.download('wordnet')
-----------------------------------------------------------------------
Program Description:
-----------------------------------------------------------------------
1.PorterStemmer implementation from nltk is used for stemming.
2.wordnet implementation from nltk is used for lemmatizing.
3.In Index_Version1.compressed,dictionary compressed using a blocked compression with k=8 and for the posting file gamma encoding is used for the gaps between document-ids also for total term frq and doc freq. While posting list compression, max-tf & doc-len are ignored. 
4.3.In Index_Version2.compressed,dictionary compressed using a front coding with k=8 (for sorted terms with long common prefix)and for the posting file delta encoding is used for the gaps between document-ids also for total term frq and doc freq. While posting list compression, max-tf & doc-len are ignored.
